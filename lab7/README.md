Пример Django проекта из Лекции #9
==================================

Лента пользователя в социальной сети.


Как запустить?
--------------

1. Ставим django

    ```
    pip install django
    ``` 
    
2. Клонируем репозиротий

    ```
    git clone git@github.com:r-dmv/lecture9.git
    ```
   
    
3. Запускаем

    ```
    python manage.py runserver
    ```
    
4. Открываем http://127.0.0.1:8000/
